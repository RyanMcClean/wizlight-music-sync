The Project

Bulb bop is a web application designed to manage and control wiz smart bulbs and sync them up with your music.


This project has been the work of Ryan Urquhart, a student at Queen's University Belfast, as his final year project.


The goal of this project is to create a web application that allows users to control their smart bulbs and sync them with their music, providing a unique and immersive experience.
